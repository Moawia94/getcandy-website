How does it work?

You will be able to use GetCandy just like any other Laravel application. You can connect to the RESTful API from your own Laravel online store application, or any other online application, e.g. iOS app. The integrations and customisations are endless. And there's no bloatware here, just use the bits you need and keep your app running lightning fast.

Core API

The API provides all the core functionality you'll need to make an amazing online store. And with plugins, you'll be able to add further functionality to fit your needs.

Admin Hub

The Admin Hub allows you to manage your catalogue, customers and orders easily. And of course plugins will enable you to add new functionality and screens with ease.

Responsive Online Store

Use our quick-start example store to get up and running quickly, or if you'd rather - code something from scratch. You don't even need to use PHP if you'd rather not...

Warehouse System

If you have a backoffice system which manages your products, inventory, orders, etc. you could integrate with GetCandy to synchronise your data.

Accounting Package

Want all your orders to automatically be fed into your accounting package? No problem, create a connector to connect to the GetCandy API!

Social Media

Why stick to just selling on a website? With GetCandy's API you can build whatever you want. Facebook shop? No problem!

Returns Management

Those pesky returned items! Easily issue refunds from the Admin Hub, build your own system/plugin or keep an eye out for an official plugin in the future.

Mobile App

Your customers don't just shop on the web, they want apps. No problem! Get your awesome iOS + Android developers to connect to your GetCandy API.
